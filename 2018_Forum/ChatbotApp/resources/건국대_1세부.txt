[이종 하이브리드 지원을 위한 컨테이너 중개 및 이동성 연구]

"CLAIR CHATBOT"

- 행사 안내 챗봇

SNS 서비스를 활용하므로 친구등록을 통해 언제 어디서든 행사 내용을 대화형으로 받아볼 수 있음

- 사진 질의 시스템

기존의 텍스트 기반 질의가 아닌 행사장 부스, 판넬 등을 촬영해 해당 내용을 받아볼 수 있음

- CLAIR APIs

CLAIR 플랫폼 상의 SaaS와 기본 서비스 간의 연동을 위한 REST 방식의 API

- 마이크로서비스 구조의 모듈 연동

각 서비스 모듈은 컨테이너 기반으로 동작, 독립적인 컴퓨팅 환경을 보장하며 REST API를 이용한 통신으로 외부 컨테이너와의 연동이 용이

- 클라우드 기반 시스템

서비스에 필요한 하드웨어, 플랫폼, 라이브러리, 개발환경을 클라우드 상에서 구축하여 빠른 개발 및 유연성, 확장성, 로드밸런싱을 제공

"Misuse Detection Based IDS With Machine Learning"

∙ A.I 를 결합한 오용 탐지 기반의 침입탐지시스템 구현도
∙ 침입 패턴 탐지를 자동화하고 고속화 함으로써 새로운 패턴의 공격에 대비할 수 있는 오용 탐지 기반의 침입탐지시스템을 설계 
∙ 네트워크 상의 패킷을 Sniffing 하고 전처리 과정을 거쳐 저장한 다음 침입 여부를 분석
∙ 패킷을 침입 데이터 패턴이 저장된 Blacklist와 정상 데이터 패턴이 저장된 Whitelist와 비교한 새로운 유형의 패턴일 경우 기계학습 알고리즘을 사용하여 Packet을 학습하고 구분
∙ 구분된 패킷이 새로운 유형의 침입 데이터이면 Blacklist 로, 정상 데이터이면 Whitelist 로 제작
